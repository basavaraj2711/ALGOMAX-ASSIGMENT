# Smart Grocery Shopping Assistant

A web-based grocery shopping assistant that helps users manage their shopping lists, track budgets, and organize items by category. It includes voice command features, customizable themes, and a store map for efficient shopping.

## Features

- **Create and Manage Shopping Lists**: Add items, set budgets, and categorize products.
- **Voice Commands**: Use voice commands to add items to your list or search for products.
- **Store Map**: View a store map to assist with navigation while shopping.
- **Budget Tracking**: Set a shopping budget and track how much you have remaining.
- **Dark Mode**: Toggle between light and dark modes for better visibility.
- **Share List**: Share your shopping list via WhatsApp or Email.
- **Responsive Design**: Fully responsive for mobile and desktop use.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/smart-grocery-shopping-assistant.git
   ```

2. Open the project directory:
   ```bash
   cd smart-grocery-shopping-assistant
   ```

3. Open the `index.html` file in your browser to get started.

## Tech Stack

- **HTML5**: Structure and content of the web page.
- **CSS (Tailwind CSS)**: For styling and responsive design.
- **JavaScript**: For app functionality (dynamic content, voice commands, etc.).
- **Font Awesome**: For icons.
- **Canvas**: For rendering the store map.

## Usage

### Setting a Budget
1. Enter the desired budget in the input field located at the top of the **Shopping List Screen**.
2. Click the **Set** button to save the budget.

### Adding Items
1. Click the **+** button on the bottom-right corner to open the **Add Item Modal**.
2. Enter the item name, price, category, and select the shopping list.
3. Click **Add Item** to add the item to your shopping list.

### Voice Command
1. Click the **Microphone** button to enable voice commands.
2. Speak a command like "Add milk" or "Where is the cereal?".
3. The app will process the voice input and act accordingly.

### Sharing Your List
1. Click on the **Share List** button.
2. Copy the shareable link or use the WhatsApp or Email buttons to share your list.

### Toggle Dark Mode
1. Go to the **Settings Screen**.
2. Use the toggle to enable or disable dark mode.

## Screens

### Shopping List Screen
- View and manage the current shopping list.
- Set a budget and track remaining funds.
- Browse items by category or search.

### Lists Screen
- View all your created lists.
- Add new shopping lists.

### Settings Screen
- Toggle between light and dark modes.
- Customize app settings.

## Modals

- **Add Item Modal**: For adding new items to the shopping list.
- **Add List Modal**: For creating new shopping lists.
- **Voice Command Modal**: For listening to and processing voice commands.
- **Share List Modal**: For generating and sharing list links.
- **Confirmation Modal**: For confirming actions like deletion.
